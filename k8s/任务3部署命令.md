## 1. 上传 YAML 到 CloudShell

将本地 `k8s` 目录中的以下文件上传到 CloudShell：

```text
configmap-secret.yaml
deployment.yaml
service.yaml
```

## 2. 应用 YAML

```bash
kubectl apply -f configmap-secret.yaml
kubectl apply -f deployment.yaml
kubectl apply -f service.yaml
```

或者如果三个文件在同一个目录：

```bash
kubectl apply -f .
```

## 3. 查看资源

```bash
kubectl get configmap
kubectl get secret
kubectl get deployments
kubectl get pods -o wide
kubectl get svc
```

## 4. 等待 Pod Running

```bash
kubectl get pods -w
```

等待出现：

```text
backend-xxxxx   1/1   Running
backend-xxxxx   1/1   Running
redis-xxxxx     1/1   Running
```

## 5. 获取 ELB 公网 IP

```bash
kubectl get svc backend-svc
```

等待 `EXTERNAL-IP` 出现公网 IP。

## 6. 访问接口

```bash
curl http://<EXTERNAL-IP>/api/ping
```

预期返回类似：

```json
{"ping_count":1,"redis":"ok","service":"flask-backend","status":"ok"}
```
